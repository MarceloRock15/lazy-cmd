# lazy-cmd

Ferramenta em Python que simplifica comandos comuns do terminal,
oferecendo atalhos interativos para tarefas frequentes.

## Objetivo
Reduzir a necessidade de memorizar comandos longos ou repetitivos,
aumentando a produtividade no uso do terminal.

## Funcionalidades (em progresso)
- [x] Comando básico funcional
- [ ] Listagem de diretórios
- [ ] Limpeza de cache/temp
- [ ] Atalhos para git
- [ ] Atalhos para processos comuns

## Tecnologias
- Python 3
- Biblioteca padrão (os, subprocess, etc.)

## Como executar
```bash
python lazy_cmd.py
